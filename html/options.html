Dual Timer

About

This is a simple dual timer extension. The idea is to use this for time-boxing. More on the time-boxing: Wikipedia

The Code?

All on GitHub. Here is the link The code is not particularly good, but was a learning experience. Maybe try it on your own to make it better! If you do, let me know and I'll update it. I hope it serves your needs.

How to Use

Set both timers to a positive integer from 1 to 99. Do the same for rounds.
